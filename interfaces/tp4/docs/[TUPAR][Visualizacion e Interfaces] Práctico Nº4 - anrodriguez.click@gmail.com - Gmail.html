Práctica 4: CSS3

 1)      Para el HTML propuesto generar un layout con distintos formatos, variando por ejemplo:

a.       Nro. de columnas de texto,

b.      bordes y posición en imágenes,

c.       diferente formato a títulos y cuerpo del texto,

d.      diferentes fondos (Color sólido, Gradiente, Imágenes)

2)      Utilizando CSS, aplicar diferentes filtros (negativo, brillo, sepia, etc.) a una imagen cargada desde disco o URL.

3)      Aplicar un efecto de transición a un elemento del HTML cuando el mouse pasa por encima, cambiando :

a.       Color
